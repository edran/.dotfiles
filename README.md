Work in progress.

# Installation
* `$ git clone --recursive git@github.com:edran/.dotfiles`
* `$ ./.dotfiles/install.sh`
* Download both IntelliJ and CLion and put them in /bin
  (For lolz, just in case)
* `$ pip install thefuck`
