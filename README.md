# Dotfiles

Compatible with `Ubuntu 14.04`. Installation is based on ansible.

# Installation
* `$ git clone --recursive git@github.com:edran/.dotfiles`
* `$ cd .dotfiles/install
* `$ sudo ./install.sh`
